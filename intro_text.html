Welcome to Fishplot Interactive

An interactive platform for dynamic clonal evolution visualization and longitudinal patient data exploration.

Fishplot Interactive is an intuitive and reproducible visualization tool designed to explore clonal dynamics over time. It allows to rapidly generate fishplots, mutation trees, and quantitative matrices from structured patient data, without requiring programming expertise.

The application supports longitudinal analysis of mutation frequencies and enables simulation of treatment impact on clonal evolution, facilitating interpretation, communication, and hypothesis generation.

Preprint available here: https://doi.org/10.1101/347534

What you can do with Fishplot Interactive:

  • Load a structured JSON file containing patient sample IDs, mutation names, node hierarchy, and clonal proportions.
  • Automatically generate fishplots, mutation trees, raw data tables, and cumulative clonal matrices.
  • Explore clonal evolution longitudinally across timepoints.
  • Simulate and visualize the impact of therapeutic interventions on mutation dynamics.
  • Modify patient data interactively and instantly recompute all visual outputs.
